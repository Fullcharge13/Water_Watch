# API Key Setup Instructions

## 1. KMA API Key Setup
   1. Visit the KMA website and navigate to the API section.
   2. Create an account or log in to your existing account.
   3. Apply for an API key by filling out the required form.
   4. Once approved, check your email for the API key.
   5. Store the API key securely, as you'll need it for authentication in your project.

## 2. WQ API Key Setup
   1. Go to the WQ API portal.
   2. Sign up or log in to your account.
   3. Request an API key through the API management page.
   4. After approval, the API key will be sent to your registered email.
   5. Keep the API key secure for future use.

## 3. Kakao API Key Setup
   1. Access the Kakao Developers site.
   2. Register your application if you haven't done so.
   3. Navigate to the 'App Settings' and find the API Keys section.
   4. Copy the REST API Key provided in this section.
   5. Remember to secure your API key as it is essential for API access.

## 4. Claude API Key Setup
   1. Visit the Claude API documentation page.
   2. Sign in or create an account.
   3. Follow instructions to generate a new API key under your account settings.
   4. The API key will be displayed on your dashboard; make sure to copy it.
   5. Store this key safely to use in your applications.

---
Make sure to replace any placeholders with your actual API keys where required in your project code.
